A homebrewed cyber threat intelligence solution for DokuWiki

Sapho was built as an off hours project to manage intelligence developed from computer network defense activities and third party sources. Building up on the considerable resources of DokuWiki Sapho automatically generates a framework of wiki resources for capturing and analyzing cyber threat intelligence and responding.

Dependencies

  • Python - Developed on Python 2.7
  • DokuWiki - Developed on 2011-05-25a "Rincewind"
  • DokuWikiXMLRPC
While not required installing the Discussion Plugin is highly recommended.

Install

  • Follow DokuWiki installation instructions found at here or try BitNavi's DokuWiki appliance
  • Enable XMLRPC interface and an account
  • Run templater.py as "python templater.py -w <wiki url> -u <username> -p <password> --setup" to create basic pages
